上岛 Blog Island

I am running an personal blog collection website, Blog island, to link amazing people together. It’s not an efficient way to find useful info, but a good way to find interested people.

If u wanna join pls comment like below:

- BI, [ur blog name], [ur blog type],[introduction with in ten words], [ur address],  [ur avator address]

岛屿，是我搭建的收藏个人博客的导航网站，现在已经收集了很多有意思的人的博客。

想要加入岛屿的同学也可以在此留言。

留言格式：

- 岛屿，[站名]，[类型]，[介绍（10 字 左右）]，[地址]，[头像地址]

征友 Friend Links

There are 7 position available for friend links in the footer of this site. If u wanna jion, pls add my website (zkpeace.com) in your blog first, and then comment ur website’s link and name below.

本站改版之后在页脚开放了7个友链位置，想要添加友链的小伙伴，可以在下方留言，如果友链没问题，不一定会添加到 footer 栏，但是一定会被添加到岛屿页面的友情链接栏。

入驻要求:

- 有眼缘，喜欢技术，生活，设计类博客。
- 已添加本站点：Zkpeace，zkpeac.com

- Footer，[站名]，[地址]。
